## Reporting a Vulnerability

If you have discovered a security vulnerability in Elevate, please report it immediately.

Please contact the core team directly by emailing security@elevate-app.example.com (or contacting Duskdynamics/Brihit Nath directly). We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress.

If the vulnerability is accepted, we will immediately work on a patch and notify users if any data was affected.
